# Smart-Dustbin-using-IOT
Developed IoT-enabled smart waste monitoring system with ESP32, IR &amp; ultrasonic sensors for fill-level tracking and automated lid control. Enabled real-time cloud dashboard (Blynk) for live monitoring, optimizing waste management and demonstrating sensor-to-cloud pipelines for smart facilities.
